What is an HTTP status code?
* Status codes are issued by a server in response to a client's request made to the server. Often found on APIs but can sometimes be displayed in the browser in cases of error codes




What port is HTTP served over?
* Port 80
* HTTPS served over 443
* Port 80 is a less secure network
* When you open any webpage, it simply sends a request as examplesite.com:80 to which browser is connecting to




What is a favicon?
* An icon associated with a particular website, and typically displayed in the address bar of a browser viewing the site.
* From the original filename used by Internet Explorer, "favicon.ico", an abbreviation of favorite followed by icon.
